## 📝 录入规则

1. 按批次从小到大录入
2. 每次录入约 20-30 条，按省份分组
3. 初始录入只填核心字段 + batch 字段：
   - id, name, level, batch, category, province, city, district, era
4. 每次录入后 commit + push 到 GitHub
5. 更新本文件记录进度
6. 所有批次基础数据完成后，再逐条完善文化背景（description, historical_value 等）
